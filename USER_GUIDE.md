# ส่วนที่ 1: คู่มือพนักงาน

## เข้าสู่ระบบครั้งแรก

1. เปิดลิงก์ **https://live-beeative.vercel.app** บนมือถือ
2. จะเห็นรายชื่อพนักงานทั้งหมด — **กดที่ชื่อตัวเอง**
3. ระบบจะให้ **ตั้ง PIN 4 หลัก** (ครั้งแรกเท่านั้น)
   - กรอก PIN ที่ต้องการ → กรอกอีกครั้งเพื่อยืนยัน
   - จำ PIN ไว้ด้วย — ใช้ Login ทุกครั้ง

> **เข้าระบบครั้งต่อไป:** กดชื่อตัวเอง → กรอก PIN 4 หลัก → เข้าได้เลย

---

## หน้าหลัก — กรอกยอดขาย

### ยอดขายของฉัน (การ์ดสีขาวด้านบน)
แสดงสรุปผลงานของตัวเองในเดือนนี้:
- ยอดวันนี้ / ยอดรวมเดือนนี้
- อันดับเทียบกับทีม (🥇 🥈 🥉)
- Incentive โดยประมาณที่จะได้
- Progress bar เทียบกับเป้าหมาย

### ตารางไลฟ์ (📅 ตารางไลฟ์ของฉัน)
- เปิดอัตโนมัติถ้ามีตารางวันนี้
- กด **🗓️** เพิ่มเข้า Google Calendar
- กด **📥** ดาวน์โหลดไฟล์ .ics (ใช้กับปฏิทิน iPhone / Android)
- กด **📥 ดาวน์โหลดทั้งหมด** ได้ตาราง 30 วันข้างหน้า

### รายการล่าสุด (📋)
- ดูรายการที่กรอกไปล่าสุด (48 ชั่วโมง)
- กดแก้ไขได้ภายใน **24 ชั่วโมง** หลังกรอก
- หลัง 24 ชั่วโมง จะขึ้น "หมดเวลาแก้ไข — กรุณาติดต่อเจ้าของช่อง"

---

## วิธีกรอกยอดขาย

1. **เลือกวันที่** — ปกติเป็นวันนี้อัตโนมัติ (ย้อนหลังได้ไม่เกิน 24 ชั่วโมง)
2. **เลือกช่วงเวลาไลฟ์** — กดเลือก Session ที่ไลฟ์ หรือ "กำหนดเวลาเอง"
3. **เลือก Platform** — TikTok / Shopee / Facebook / อื่นๆ
4. **กรอกยอดขาย** — ใส่ตัวเลขยอดขาย (บาท)
5. **หมายเหตุ** — ไม่บังคับ เช่น "มีโปรโมชั่น", "ยอดรวม 2 platform"
6. กดปุ่ม **✅ บันทึกยอดขาย**

---

## โปรไฟล์ของฉัน

กดที่รูปโปรไฟล์ (มุมบนซ้าย) หรือไปที่ https://live-beeative.vercel.app/profile

- **เปลี่ยนรูปโปรไฟล์** — กด 📷 แล้วเลือกรูปจากมือถือ
- **เปลี่ยน PIN** — กรอก PIN ใหม่ 4 หลัก → ยืนยันอีกครั้ง
- กด **💾 บันทึก**

---

# ส่วนที่ 2: คู่มือเจ้าของ

## เข้าสู่ระบบ

ไปที่ https://live-beeative.vercel.app/owner/login

## เมนูหลัก (แถบด้านล่าง)

| เมนู | ใช้ทำอะไร |
|---|---|
| 📊 ภาพรวม | Dashboard ยอดขายรวม, รายการล่าสุด |
| 📅 ตาราง | จัดตารางไลฟ์ให้พนักงานแต่ละคน |
| 🎯 เป้ายอด | ตั้งเป้ายอดขายและงบรายจ่าย |
| 👥 พนักงาน | เพิ่ม/แก้ไข/จัดการพนักงาน |
| 🏖️ วันลา | จัดการสิทธิ์วันลาพนักงาน |
| 💰 การเงิน | รายรับ-รายจ่าย, export รายงาน |
| 📈 รายงาน | วิเคราะห์ยอดขายเชิงลึก |

---

## 📊 Dashboard (ภาพรวม)

- เลือกดูยอดขาย **วันนี้ / สัปดาห์นี้ / เดือนนี้**
- ดู Leaderboard พนักงาน พร้อม progress bar
- ดูยอดขายแยกตาม Platform
- ดูรายการล่าสุด 5 อันดับ
- **อัปเดตอัตโนมัติทุก 30 วินาที** — ไม่ต้อง refresh เอง
- ไอคอน 📊 ในเมนูล่างมีตัวเลขแดง = จำนวนรายการที่กรอกวันนี้

---

## 📅 ตาราง (จัดตารางไลฟ์)

**วิธีเพิ่มตาราง:**
1. เลือกพนักงาน (กรองด้านบน)
2. เลือกวัน → กรอกเวลาเริ่ม-สิ้นสุด → หมายเหตุ (ถ้ามี)
3. กด **เพิ่มตาราง**

**ดูตามสัปดาห์:** กดลูกศร ‹ › เพื่อไปสัปดาห์ก่อน/หน้า

พนักงานจะเห็นตารางนี้บนหน้าแอปของตัวเองโดยอัตโนมัติ

---

## 🎯 เป้ายอดขาย

- **เป้ารวมทีม** — ยอดขายรวมทุกคนที่ต้องทำในเดือนนี้
- **เป้ารายคน** — ตั้งเป้าแยกให้แต่ละพนักงาน
- **งบรายจ่าย** — งบค่าใช้จ่ายที่วางแผนไว้เดือนนี้
- กด **บันทึก** → ระบบคำนวณ % ความคืบหน้าให้อัตโนมัติ

พนักงานจะเห็น progress bar เทียบเป้าบนหน้าแอปตัวเองด้วย

---

## 👥 จัดการพนักงาน

### เพิ่มพนักงานใหม่
1. กรอกชื่อ, เงินเดือน, อัตรา Incentive (%)
2. PIN — **ไม่ต้องตั้งให้** พนักงานตั้งเองในครั้งแรกที่ Login
3. กด **เพิ่มพนักงาน**

### แก้ไขข้อมูลพนักงาน
กดที่ชื่อพนักงาน → เข้าหน้ารายละเอียด:
- แก้ชื่อ / เงินเดือน / อัตรา Incentive
- Override PIN (กรณีพนักงานลืม PIN)
- เปลี่ยนสถานะ Active/Inactive
- ดูสถิติ: ยอดขายรวม, จำนวนรายการ, Incentive ที่ได้
- ดูสิทธิ์วันลาของพนักงานคนนั้น
- **ลบพนักงาน** (กดปุ่มสีแดงด้านล่าง — ข้อมูลยอดขายยังเก็บไว้)

### Badge "ยังไม่ตั้ง PIN"
พนักงานที่ยังไม่ได้ตั้ง PIN จะมีป้าย สีส้ม — รอให้พนักงาน Login ครั้งแรกเอง

---

## 🏖️ วันลา

**ตั้งสิทธิ์วันลาให้พนักงาน:**
1. เลือกพนักงาน (แท็บด้านบน)
2. กรอกจำนวนวันลาแต่ละประเภท (ลาป่วย / ลากิจ / ลาพักร้อน / อื่นๆ)
3. กด **บันทึกสิทธิ์**

**บันทึกวันลาที่ใช้:**
- เลือกพนักงาน → เลือกประเภทลา → วันที่ → จำนวนวัน → กด **บันทึก**

**โชว์/ซ่อนข้อมูลวันลาให้พนักงาน:**
- กดปุ่ม **"โชว์"** ที่ชื่อพนักงาน → พนักงานจะเห็นสิทธิ์ลาตัวเองบนแอป
- กดอีกครั้งเพื่อซ่อน (เจ้าของเห็นอยู่เสมอ)

---

## 💰 การเงิน

**ดูรายรับ-รายจ่าย:**
- เลือกเดือน (กดที่ช่องเดือน มุมขวาบน)
- ดูกำไร/ขาดทุนสุทธิ
- ดูสัดส่วนรายจ่ายแยกหมวดหมู่
- ดู Leaderboard พนักงาน พร้อมเงินเดือน + Incentive

**เพิ่มรายจ่าย:**
1. กด **➕ เพิ่มรายจ่าย**
2. กรอกชื่อ, จำนวนเงิน, วันที่, หมวดหมู่
3. ถ้าเป็น **รายจ่ายประจำ** (เช่น ค่าเช่า ค่าน้ำไฟ) ติ๊ก "รายจ่ายประจำ"
4. กด **บันทึกรายจ่าย**

**Export รายงาน:**
- กด **📊 Export Excel** → ไฟล์ .xlsx
- กด **📄 Export PDF** → ไฟล์ PDF พร้อมพิมพ์

## 📈 รายงาน

วิเคราะห์ยอดขายเชิงลึก เลือกดูแบบ:
- **ตาม Session** — เปรียบเทียบยอดแต่ละช่วงเวลาไลฟ์
- **รายวัน** — ยอดแต่ละวัน
- **รายสัปดาห์** — ยอดแต่ละสัปดาห์
- **รายเดือน** — ยอดแต่ละเดือน

กรองตาม Platform และช่วงวันที่ได้

## คำถามที่พบบ่อย

**Q: พนักงานลืม PIN ทำอย่างไร?**
A: เจ้าของไปที่ 👥 พนักงาน → กดชื่อพนักงาน → กรอก PIN ใหม่ให้ → บันทึก

**Q: ต้องการเพิ่มช่วงเวลาไลฟ์ใหม่ (Session)?**
A: ไปที่ /owner/sessions → กด "เพิ่ม Session"

**Q: ยอดขายไม่แสดงบน Dashboard?**
A: รอ 30 วินาที ระบบ refresh อัตโนมัติ หรือกด refresh บน browser

**Q: Export Excel ไม่ได้?**
A: ไปที่หน้าการเงิน (💰) → เลือกเดือน → กด Export Excel

**Q: พนักงานลาออก ทำอย่างไร?**
A: ไปที่ 👥 พนักงาน → กดชื่อ → กด "ลบพนักงาน" — ข้อมูลยอดขายยังอยู่ในระบบ
